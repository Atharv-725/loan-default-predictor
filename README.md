\# Loan Default Predictor 🏦



A machine learning project to predict whether a loan applicant will default, 

using real-world style financial features.



\## Models Used

| Model | Accuracy |

|-------|----------|

| Logistic Regression | \~85% |

| Random Forest | \~88% |



\## Features

\- Income, Age, Loan Amount

\- Credit Score, Employment Years

\- Debt Ratio, Credit Lines, Payment History



\## Tech Stack

Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Jupyter Notebook



\## How to Run

```bash

pip install pandas scikit-learn matplotlib seaborn jupyter

jupyter notebook loan-default-predictor.ipynb

```



\## Key Findings

\- Random Forest outperforms Logistic Regression

\- Credit score and payment history are the most important features
